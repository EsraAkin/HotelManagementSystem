------------------------------------ÖN HAZIRLIK-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1) Intelij Idea'da Yeni bir proje oluştur.

2) pom.xml klasörüne gel. Dependenciesleri ekle. (Hybernate ve jdbc-core için vs. İnternetten maven repocitoryden bulabilirsin.) hibernate.cfg.xml

3) resources-->hibernate.cfg.xml--> klasöründe gel. Database connection settings ve Other Hibernate configuration properties ayarlamalarını yap hibernate.cfg.xml

4) PostgreSql de Database oluştur. DİKKAT: hibernate.cfg.xml dosyasındaki ile aynı isimde olmasına dikkat et.

5) java klasörü sağ click-->com.tpe.hotelManagementSystem-->gibi package ismi tanımla. (İçiçe de yazabilirsin tek tek.)

6) Katmanlı mimariye göre packageleri oluştur. Her bir işlev için bir package oluşturmalısın.
config → Uygulama yapılandırma ayarları (örneğin, Hibernate, Spring, DB bağlantı ayarları).
exception → Özel hata sınıfları ve uygulamaya özgü istisnalar.
main → Uygulamanın çalıştırıldığı ana sınıf (örneğin, main metodu içerir).
model → POJO'lar, Entity sınıfları ve veri modelleri.
repository → Veritabanı işlemleri (CRUD) için DAO veya JPA sınıfları.
service → İş mantığı (business logic) ve iş kurallarını içeren servis sınıfları.
